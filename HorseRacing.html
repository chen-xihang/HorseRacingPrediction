2 Overall summary

This project develops a race-level win probability model for horse racing using a conditional logit framework that accounts for the competitive structure within each race. The model incorporates performance ratings for horses, jockeys, and trainers, alongside race-level features (distance, going, field size, race type), and tests whether these variables improve prediction after accounting for Betfair Starting Price (BSP). Performance is evaluated out-of-sample using probabilistic metrics (log loss, Brier score, calibration) and simulated betting returns with confidence intervals, with a focus on economically meaningful residual edge.

Missingness is therefore outcome-dependent rather than random, reflecting structural censoring. These variables are excluded from predictive modelling to avoid leakage and bias.

4 Implementation of Rating Systems

I constructed a forward-looking rating system for each entity (i.e., horses, jockeys and trainers) separately using RPR. For each entity at each race, I aggregated only prior runs and computed a time-decayed weighted mean of past ratings, thus recent performances contribute more than older ones. I then applied empirical Bayes-style shrinkage towards the global ratings. This ensures ratings were stable, leakage-safe, and avoids overreacting to small sample sizes.

Code 
df <- df |>
  shrinkage_rating(
    id_col = "horse_id",
    out_col = "horse_rating"
  ) |>
  shrinkage_rating(
    id_col = "jockey_id",
    out_col = "jockey_rating"
  ) |>
  shrinkage_rating(
    id_col = "trainer_id",
    out_col = "trainer_rating"
  )

Mathematically speaking, this rating system is done via \[R_e(t) = \frac{W_e(t)}{W_e(t) + k} \cdot \bar{r}_e^{(\text{decay})}(t) +\frac{k}{W_e(t) + k}\mu\] where \[\bar{r}_e^{(\text{decay})}(t) = \frac{\sum_{i: t_i < t} \exp(-\lambda (t - t_i)) r_{e,i}} {\sum_{i: t_i < t} \exp(-\lambda (t - t_i))}\] is the time-decayed rating and associated terms are defined as follows:

  • \(R_e(t)\) is the rating of entity \(e\) (horse, jockey, or trainer) at time \(t\).
  • \(r_{e,i}\) is the observed rating from past run \(i\) of entity \(e\).
  • \(t_i\) is the date of past run \(i\), and \(t\) is the current race date.
  • \(\bar{r}_e^{(\text{decay})}(t)\) is the exponentially time-decayed weighted mean of prior performances.
  • \(W_e(t) = \sum_{i: t_i < t} \exp(-\lambda (t - t_i))\) is the effective sample size (total decayed weight).
  • \(\mu\) is the global mean rating across all entities.
  • \(k\) is the shrinkage parameter controlling the strength of pull toward the global mean.
  • \(\lambda = \frac{\ln(2)}{h}\) is the decay parameter, where \(h\) is the half-life in days, corresponding to half_life_days in the function.

5 Building Models

There are three models to estimate win probabilities:

  • A market-only benchmark based on ltp_5min (model_m0), treating this as a strong baseline.
  • A feature-based model using entity ratings and race characteristics (model_m1).
  • A combined model that conditions on market-implied probabilities to test for incremental predictive signal.

All models are trained chronologically and evaluated strictly out-of-sample using log loss, Brier score, and simulated betting performance.

Code 
# Train-test split
df <- df |>
  dplyr::arrange(date, race_time, race_id) |>
  dplyr::filter(!is.na(ltp_5min))

race_order <- df |>
  distinct(race_id, date, race_time) |>
  dplyr::arrange(date, race_time, race_id) |>
  dplyr::mutate(race_index = row_number())

train_cutoff <- floor(0.7 * nrow(race_order))

train_races <- race_order |>
  dplyr::filter(race_index <= train_cutoff) |>
  dplyr::pull(race_id)

df <- df |>
  dplyr::mutate(
    set = if_else(race_id %in% train_races, "train", "test")
  )

train_df <- df |> filter(set == "train")
test_df  <- df |> filter(set == "test")

train_df <- train_df |>
  mutate(
    p_market = 1 / obs_bsp, 
    logit_market = log(p_market / (1 - p_market))
  )

test_df <- test_df |>
  mutate(
    p_market = 1 / ltp_5min,
    logit_market = log(p_market / (1 - p_market))
  )

model_m0 <- glm(
  obs_is_winner ~ logit_market,
  data = train_df,
  family = binomial()
)

model_m1 <- glm(
  obs_is_winner ~ 
    horse_rating +
    jockey_rating +
    trainer_rating +
    ns(age, 3) +
    ns(carried_weight, 3) +
    ns(race_distance, 3) +
    race_type_simple +
    going_clean +
    n_runners,
  data = train_df,
  family = binomial()
)

model_m2 <- glm(
